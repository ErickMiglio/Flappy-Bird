# Flappy Bird 🐦
Um clássico jogo Flappy Bird reconstruído com tecnologias web modernas!
## Tecnologias Utilizadas
- **HTML5** - Estrutura do jogo
- **CSS3** - Estilização com gradientes e animações
- **JavaScript (ES6+)** - Lógica do jogo
- **Canvas API** - Renderização gráficos 2D
## Como Jogar
1. Clique no botão **JOGAR** ou pressione **ESPAÇO** para iniciar
2. Pressione **ESPAÇO** ou **clique na tela** para fazer o pássaro pular
3. Desvie dos canos verdes!
4. A cada 10 pontos, a velocidade aumenta
5. Seu recorde é salvo automaticamente no navegador
## Jogar Online
🔗 **https://flappy-bird-erickmiglio.vercel.app/**
---
Divirta-se! 🎮
